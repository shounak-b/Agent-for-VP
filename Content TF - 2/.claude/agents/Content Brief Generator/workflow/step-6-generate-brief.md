# Step 6: Generate Content Brief

**Goal:** Synthesize all collected research and insights into a comprehensive, actionable content brief that includes strategically structured headlines, internal linking opportunities, target word count, content goal, and meta description.

---

## Why This Matters

This is the deliverable that guides content creation. A well-structured content brief ensures the landing page ranks for target keywords, aligns with brand guidelines, provides value to the target audience, and integrates seamlessly into the site architecture.

---

## Review All Collected Information

Before generating the brief, ensure you have:
- **Page Name** (from Step 1)
- **Primary Keyword** (from Step 1)
- **Secondary Keywords** (from Step 1)
- **Client Name** (from Step 1)
- **Page Content Brief Guide** (from Step 1)
- **Domains** (from Step 1)
- **Recommended Word Count Range** (from Step 2)
- **Page Level** (from Step 3)
- **Internal Linking Opportunities** (from Step 3)
- **Existing Page Headlines** (from Step 5, if available)
- **Competitor Headlines** (from Step 6)

---

## Instructions

### A. Generate Strategic Headline Structure

Create SEO-optimized headlines following these requirements:

**SEO Hierarchy:**
- **1 H1 heading** — Must include the Primary Keyword and describe the main topic
- **H2 headings** — Subtopics of the H1; major content sections
- **H3 headings** — Subtopics of the parent H2
- **Character limit** — All headings must be 60 characters or fewer
- **Logical flow** — Must follow a readable progression

**Content Strategy:**
- Analyze competitor headlines from Step 6 for effective patterns and gaps
- Ensure important topics from existing page (Step 5) aren't omitted
- Incorporate Secondary Keywords naturally into H2/H3 headings
- Align with Page Content Brief Guide and target audience needs
- Consider the client's messaging pillars as defined in [../brand-guidelines.md](../brand-guidelines.md)

See [headline-strategy.md](headline-strategy.md) for detailed headline creation guidance.

---

### B. Integrate Internal Linking Opportunities

Reference the relevant pages from Step 3. For each internal link, specify:
- **Anchor text** — from Step 3
- **URL** — from Step 3
- **Placement** — Which headline section the link appears under

Distribute links across multiple sections for balanced link equity.

**Example:**
```
Under H2: Key Features and Capabilities
- Link to "Advanced Reporting Tools" using anchor text "reporting and analytics"
```

---

### C. Apply Brand Guidelines

**CRITICAL:** All headlines MUST comply with Visual Planning's brand guidelines.

Reference: [../brand-guidelines.md](../brand-guidelines.md)

**Scan all headlines for prohibited terminology:**
- **Absolutes/Guarantees**: Avoid "ensure", "guarantee", "always", "all", "any", "100%", "effortless", "seamless", "eliminates"
- **Superlatives**: Avoid "best", "best-in-class", "most powerful", "revolutionary", "game-changing", "cutting edge", "unique", "unmatched", "perfect"
- **Competitive overstatements**: Avoid "kills spreadsheets", "destroys the competition", "replaces all other tools"
- **Vague terms**: Avoid "powerful", "smart", "robust", "expertise", "solution" (without specifics)

**Check all tense variations**: Present, past, future, plural forms (e.g., "ensures", "ensured", "ensuring")

**Replace with compliant alternatives:**
- "Ensure" → "Helps ensure", "Designed to support", "Supports"
- "Best" → "Among the leading options", "A strong choice for"
- "Perfect" → "Well-suited for", "Designed for"
- "Eliminate" → "Reduce", "Minimize", "Help prevent"
- "Seamless" → "Streamlined", "Well-integrated"

**Apply tone of voice:** Clear, practical, empathetic, outcome-focused, down-to-earth — never boastful or vague

---

### D. Define Target Word Count Range

Use the recommended word count range from Step 2.
- Ensure minimum floor: 500 words
- Ensure maximum ceiling: 1500 words
- Consider page level and content depth

**Example:** `Target Word Count: 800-1200 words`

---

### E. Create Content Goal

Write 2-3 future tense sentences describing what the page will accomplish.

Base the goal on:
- Generated headlines and content structure
- Page Content Brief Guide from Step 1
- Target audience (ICP) and their needs
- Primary Keyword intent

Frame from user's perspective: what will they learn, achieve, or understand?

**Example:**
```
This page will help operations managers and project managers understand how Visual Planning's resource scheduling software helps teams replace spreadsheets and manual planning with a visual, configurable platform. It will highlight key capabilities such as drag-and-drop scheduling, smart resource assignment, and real-time collaboration tools. Visitors will leave with a clear understanding of how Visual Planning fits their workflow and how to get started with a free trial or demo.
```

---

### F. Generate Page Title

Write a page title for the `<title>` tag:
- **Format:** `{Primary Keyword} | {Brand Name}`
- **Length:** 50–60 characters
- **Review top-ranking SERP title tags** for the Primary Keyword to identify modifiers worth incorporating (e.g., year, "Free Trial", product qualifiers)

**Example:** `Resource Scheduling Software | Visual Planning`

---

### G. Generate Meta Description

Write a compelling meta description:
- **Length:** 150-160 characters
- **Include:** Primary Keyword naturally
- **Summarize:** Page'˜s value proposition
- **Use:** Action-oriented, outcome-driven language
- **Apply:** Brand guidelines (avoid prohibited terminology)
- **Format:** Write as complete sentences (not fragments)

**Example:**
```
Discover resource scheduling software designed to help teams replace spreadsheets with a visual, configurable planning platform. Free trial available for teams of any size.
```

## Quality Checks

Before finalizing, verify:

**Headline structure:**
- [ ] Exactly 1 H1 that includes the Primary Keyword
- [ ] Multiple H2s as major content sections
- [ ] H3s used appropriately as subtopics under H2s
- [ ] All headings are 60 characters or fewer
- [ ] Logical flow from section to section

**Brand compliance:**
- [ ] Scanned all headlines for prohibited terms
- [ ] Checked all tense variations and plural forms
- [ ] Replaced prohibited terms with compliant alternatives
- [ ] Applied outcome-driven, straightforward tone

**Content strategy:**
- [ ] Headlines align with Page Content Brief Guide
- [ ] Important topics from existing page (if any) retained
- [ ] Secondary Keywords incorporated naturally
- [ ] Competitor insights informed content structure
- [ ] Content addresses ICP goals and pain points

**Page title:**
- [ ] Page title follows `{Primary Keyword} | {Brand Name}` format
- [ ] Page title is 50–60 characters
- [ ] Top-ranking SERP title tags reviewed for modifiers

**Internal linking:**
- [ ] 2-5 internal links specified
- [ ] Each link uses URL and anchor text from Step 3
- [ ] Each link includes placement (which section)
- [ ] Anchor text is natural and keyword-relevant

**Supporting elements:**
- [ ] Target word count range is 500-1500 words
- [ ] Content goal is 2-3 future tense sentences
- [ ] Meta description is 150-160 characters and includes Primary Keyword
- [ ] Meta description follows brand guidelines

See [common-patterns.md](common-patterns.md) for quality check templates.

## Error Handling

- If any previous step failed or returned "NA", work with available information and note limitations in NOTES section
- If brand guidelines file cannot be accessed, proceed with general SEO best practices and flag for manual review
- If no internal linking opportunities were identified in Step 4, note "Manual internal linking review needed"
