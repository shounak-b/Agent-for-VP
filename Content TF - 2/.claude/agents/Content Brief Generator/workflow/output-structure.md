## Section Guidelines

### Title & Date
- Format: `# Visual Planning | [Page Topic]: Content Brief`
- Date: M.D.YYYY (e.g., 3.6.2026)

### Goal of Content
- Bullet points, italicized
- Include: page objective, target personas, conversion goal

### Page Title
- Format: `{Primary Keyword} | Visual Planning`
- Length: 50-60 characters

### Meta Description
- Length: 150-160 characters
- Include Primary Keyword naturally

### URL Structure
- Start with `https://www.visual-planning.com/en/`
- Lowercase, hyphens for spaces

### Target Word Count
- Single number or range from Step 2

### Headings/Sections
See [Headings Format](#headings-format) below.

### Internal Links
- List each URL with anchor text as sub-bullet
- Use "URL TBD" for pages not yet created

## Headings Format

**Formatting:**
- Prefix: `**H1:**`, `**H2:**`, `**H3:**` (bold)
- Headline text: *italicized*
- CTAs: `**CTA:** [text]` on own line
- Indent: H2 under H1, H3 under H2

**CTA Placement:**
- After H1 (primary CTAs)
- After key H2/H3 sections
- Final closing H2

**Special Sections:**
- Testimonials: `[Include testimonial carousel]`
- FAQs: H2 with H3s as questions

**Example:**
```markdown
**H1:** *Scheduling software for mid-market businesses*

**CTA:** Get started for free

  **H2:** *Growth has exposed gaps in your planning*

    **H3:** *Teams expand, but visibility does not*

    **H3:** *Departments operate in silos*

  **H2:** *What our users are saying*

  [Include testimonial carousel]

  **CTA:** See more testimonials
```

**Rules:**
- 1 H1 per page with Primary Keyword
- All headlines ≤60 characters
- H3s only under H2s
- NO writing instructions (added in Step 8)
